```
├── env-example
├── nodemon.json
├── now.json
├── package-lock.json
├── package.json
└── tsconfig.json
```

## Setup the project
1. Copy env-example and replace the credentials `cp env-example .env`
2. Run `npm i` or `yarn install`
3. You can run it in dev with nodemon using `npm run dev` or simply `npm start`
